Functions support list grouped by area.

⸻

🔢 Math & Basic Calculations

These are the backbone of most spreadsheets:
	•	SUM – add a range
	•	AVERAGE – mean value
	•	COUNT – count numeric cells
	•	COUNTA – count non-empty cells
	•	MIN / MAX – smallest / largest value
	•	ROUND, ROUNDUP, ROUNDDOWN
	•	ABS – absolute value
	•	MOD – remainder

⸻

🔍 Logical Functions

Used for decision-making and conditions:
	•	IF – conditional logic
	•	IFS – multiple conditions
	•	AND, OR, NOT
	•	IFERROR – handle errors gracefully

⸻

🔎 Lookup & Reference

Critical for pulling data across tables:
	•	VLOOKUP – vertical lookup (older, still widely used)
	•	HLOOKUP – horizontal lookup
	•	XLOOKUP – modern replacement (more flexible)
	•	INDEX – return value from position
	•	MATCH – find position of a value
	•	INDEX + MATCH – powerful combo
	•	OFFSET – dynamic ranges

⸻

📝 Text Functions

For cleaning and manipulating strings:
	•	CONCAT / CONCATENATE – join text
	•	TEXTJOIN – join with delimiter
	•	LEFT, RIGHT, MID – extract substrings
	•	LEN – text length
	•	TRIM – remove extra spaces
	•	UPPER, LOWER, PROPER
	•	SUBSTITUTE – replace text
	•	FIND / SEARCH – locate text

⸻

📅 Date & Time

Very common in reporting and finance:
	•	TODAY – current date
	•	NOW – current date & time
	•	DATE – build a date
	•	YEAR, MONTH, DAY
	•	DATEDIF – difference between dates
	•	EOMONTH – end of month
	•	WORKDAY, NETWORKDAYS

⸻

📊 Statistical

Used in analysis and reporting:
	•	SUMIF, SUMIFS
	•	COUNTIF, COUNTIFS
	•	AVERAGEIF, AVERAGEIFS
	•	MEDIAN
	•	STDEV.P, STDEV.S

⸻

📈 Dynamic Array / Modern Excel (Newer)

These are increasingly important:
	•	FILTER – return matching rows
	•	SORT / SORTBY
	•	UNIQUE – distinct values
	•	SEQUENCE – generate series
	•	RANDARRAY – random arrays

⸻

💰 Financial (Common but situational)
	•	PMT – loan payments
	•	NPV – net present value
	•	IRR – internal rate of return
